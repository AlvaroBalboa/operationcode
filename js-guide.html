Newbie Guide: JavaScript

Brief history of language

JavaScript is an interpreted programming language originally created in 1995 by Brendan Eich. It is known as the language of the web, but it can also be run outside of a web browser.

Where is Javascript used? What can I program with it?

  • Browser / client-side (websites UI, single-page-application frameworks)
  • Server-side / non-browser runtimes (NodeJS)
  • Additional cross-platform applications (Electron, IoT, anything else)

Key concepts & best practices w/ Javascript

  • DOM (Document Object Model)
  • Primitive types (explain these briefly?)
  • Prototypical Inheritance
  • Closure
  • Functions as first-class citizens
  • ES6/2015 and onward
